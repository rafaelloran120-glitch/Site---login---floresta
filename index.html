✨ Curiosidades do Universo

  • O universo tem cerca de 13,8 bilhões de anos.
  • Existem mais estrelas no universo observável do que grãos de areia na Terra.
  • Buracos negros podem dobrar o espaço-tempo.
  • A Via Láctea tem mais de 100 bilhões de estrelas.
  • Galáxias podem colidir e se fundir com o tempo.

🐍 Curiosidades sobre Python

  • Python foi criado por Guido van Rossum em 1991.
  • O nome vem do grupo de comédia "Monty Python".
  • É uma das linguagens mais usadas em ciência de dados e IA.
  • Tem uma sintaxe simples, parecida com o inglês.
  • Grandes empresas como Google e Netflix usam Python.

☕ Curiosidades sobre Java

  • Java foi lançado pela Sun Microsystems em 1995.
  • O lema é "escreva uma vez, rode em qualquer lugar".
  • É muito usado em aplicativos Android.
  • É uma das linguagens mais populares do mundo.
  • Grandes sistemas bancários usam Java pela segurança.
